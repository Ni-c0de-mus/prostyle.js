The mousemove controller uses the position of the mouse on the frame to seek the player position. As the mouse is moved over the frame, the mouse's X position relative to the width of the frame is used to determine the percentage of the story that should be played.

There are start and end parameters that default to 0 and 100, representing the left edge of the frame and right edge, respectively. These edges can be adjusted to shorten the *hit* area. If the end value is less than the start, the effect will work in reverse. If the values are the same, it will cause a split point so that when the mouse is to the left of it, the story will be seeked to the beginning, and to the right to the end.

When the mouse leaves the frame, the player will begin playing like normal.

The mousemove controller is a ProStyle Plus premium extension.
